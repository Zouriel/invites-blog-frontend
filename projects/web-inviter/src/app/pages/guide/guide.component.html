The one rule: every row must have at least one of email or phone. A good example Common mistakes
  • Empty rows — a row with neither email nor phone is skipped.
  • Merged cells — flatten them; one guest per row.
  • Header typos — keep headers lowercase: email, not E-Mail.
  • Phone without country code — set a default country on the upload page, or prefix with +.
  • Duplicate contacts — the same email/phone twice is flagged as a duplicate.
  • Names in the email column — keep each value in its own column.
